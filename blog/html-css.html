The difference between Margin, Border and Padding

Margin

Margin is the area around an element that separates the element from the rest of the elements on a page.


Border

Border is the area that surrounds an element, typically for styling the border has no effect on the other elements on the page.


Padding

Padding is the area around an element that pushes elements away from the border of an element.

The box model is a way of showing the areas of a element. Each area is a layer, surrounding the content.
